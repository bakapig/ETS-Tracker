# Map train icons

Replace any SVG here with your own artwork. Keep the **filename** the same so the app picks it up automatically.

| File | Train type | Used when |
|------|------------|-----------|
| `ets.svg` | Class 93/2 ETS | Label starts with `ETS` |
| `komuter_83.svg` | Class 83 Komuter | Label starts with `EMU` |
| `komuter_92.svg` | Class 92 Komuter | Label starts with `SCS` |
| `dmu_61.svg` | Class 61 DMU | Label starts with `DMU` |
| `intercity.svg` | Intercity | ERT / ES routes |
| `default.svg` | Fallback | Unknown type |

Recommended size: **32×32 px** (SVG viewBox `0 0 32 32`).

Web: `web/public/trains/`  
Mobile: `mobile/assets/trains/` (same filenames)
